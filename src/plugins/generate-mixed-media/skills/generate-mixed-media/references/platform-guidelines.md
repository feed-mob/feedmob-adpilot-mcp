# Mixed Media Platform Guidelines

## Text Overlay Specifications

### TikTok (1080×1920, 9:16)

**Safe Zones**:
- Top: 150px margin (status bar)
- Bottom: 300px margin (UI elements)
- Sides: 40px margin

**Typography**:
- Headline: 36-48px, bold, centered
- Body: 18-24px, regular, centered
- CTA: 24-32px, bold, button style

**Colors**:
- Primary text: White with 2px black shadow
- CTA: Platform pink (#FE2C55) or brand color
- Overlay: Semi-transparent black (40-60% opacity) if needed

### Instagram Feed (1080×1080, 1:1)

**Safe Zones**:
- All sides: 60px margin
- Bottom: 120px for engagement UI

**Typography**:
- Headline: 32-40px, bold
- Body: 16-20px, regular
- CTA: 20-28px, medium weight

**Colors**:
- Primary text: White or dark based on image
- CTA: Gradient or solid accent
- Overlay: Subtle gradient from bottom (30-50% opacity)

### Instagram Story (1080×1920, 9:16)

**Safe Zones**:
- Top: 200px (username, close button)
- Bottom: 250px (reply, share)
- Sides: 40px margin

**Typography**:
- Headline: 40-52px, bold
- Body: 20-26px, regular
- CTA: 28-36px, swipe-up style

**Colors**:
- Primary text: White with shadow
- CTA: Bright accent with arrow indicator
- Overlay: Gradient or blur effect

### Facebook Feed (1200×628, 1.91:1)

**Safe Zones**:
- All sides: 50px margin
- Right: 200px for engagement buttons

**Typography**:
- Headline: 28-36px, bold, left-aligned
- Body: 14-18px, regular, left-aligned
- CTA: 18-24px, button style

**Colors**:
- Primary text: Dark (#1c1e21) on light, white on dark
- CTA: Facebook blue (#1877F2) or brand color
- Overlay: Light gradient from left (20-40% opacity)

### LinkedIn (1200×627, 1.91:1)

**Safe Zones**:
- All sides: 60px margin
- Bottom: 80px for engagement

**Typography**:
- Headline: 26-34px, bold, professional font
- Body: 14-16px, regular
- CTA: 16-22px, professional button

**Colors**:
- Primary text: Dark (#000000) or white
- CTA: LinkedIn blue (#0A66C2) or professional accent
- Overlay: Clean, minimal (10-30% opacity)

## Text Contrast Guidelines

### Light Images
- Use dark text (#111111 or #1c1e21)
- Add subtle white glow or light shadow
- Consider light overlay behind text

### Dark Images
- Use white text (#FFFFFF or #F5F5F5)
- Add black shadow (2-4px offset, 50% opacity)
- Consider dark gradient overlay

### Busy/Complex Images
- Add semi-transparent overlay (40-60% opacity)
- Use solid background behind text block
- Increase shadow intensity

## Typography Best Practices

1. **Hierarchy**: Headline > Body > CTA (clear size progression)
2. **Readability**: Minimum 14px for body, 24px for headlines
3. **Line Height**: 1.3-1.5 for body, 1.1-1.2 for headlines
4. **Letter Spacing**: Slightly increased for all-caps headlines
5. **Font Pairing**: Sans-serif for digital (Inter, Roboto, SF Pro)

## Export Specifications

| Format | Use Case | Quality |
|--------|----------|---------|
| PNG | Graphics, text-heavy | Lossless |
| JPEG | Photos, gradients | 85-95% quality |
| WebP | Web delivery | 80-90% quality |

Always export at native platform resolution (no upscaling).
